# NGPTF - Automated Penetration Testing Framework

An Automated Penetration Testing Framework developed as a Bachelor of Computer Applications (BCA) Major Project.

The framework automates multiple phases of penetration testing, including reconnaissance, information gathering, vulnerability assessment, and report generation, helping security professionals perform authorized security assessments more efficiently.

---

## Features

- Automated Reconnaissance
- Host Discovery
- Port Scanning
- Service Enumeration
- Vulnerability Assessment
- Network Information Gathering
- Report Generation
- Modular Architecture
- Command Line Interface
- Logging Support

---

## Prerequisites

Before running NGPTF, make sure the following are installed:

- Linux Operating System (Ubuntu/Kali Linux Recommended)
- Python 3.x
- Git

### External Security Tools

- Nmap
- WhatWeb
- Whois
- DNSRecon
- Nikto
- SQLMap

## Installation

### Clone the Repository

```bash
git clone https://github.com/yourusername/NGPTF-Automated-Penetration-Testing-Framework.git

cd NGPTF-Automated-Penetration-Testing-Framework
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Framework

```bash
python main.py
```
